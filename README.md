# OIBSIP - Python Programming Internship

This repository contains all the projects completed during my Python Programming Internship at Oasis Infobyte..

## Internship Tasks

### Task 1 - Voice Assistant
Developed a Python-based voice assistant that responds to voice commands and performs tasks like opening websites, telling time/date, and searching Wikipedia.

- Repository: https://github.com/Bhagatkhushi/OIBSIP/tree/main/Task%201-Voice%20Assistan

### Task 2 - BMI Calculator
Created a GUI-based BMI Calculator using Python and Tkinter that calculates Body Mass Index and classifies users into different health categories.

- Repository: https://github.com/Bhagatkhushi/OIBSIP/tree/main/Task%202-BMI%20Calculator

### Task 3 - Password Generator
Designed a secure password generator that creates strong passwords with customizable length and character combinations to improve account security.

- Repository: https://github.com/Bhagatkhushi/OIBSIP/tree/main/Task%203-Password%20Generator

### Task 4 - Weather App
Built a weather forecasting application using Python and OpenWeatherMap API to fetch and display real-time weather information for different locations.

- Repository: https://github.com/Bhagatkhushi/OIBSIP/tree/main/Task%204-Weather%20App
